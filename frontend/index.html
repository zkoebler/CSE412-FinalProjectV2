Project Description:

This project allows users to visualize data from the "bureau-of-fire-prevention-certificates-of-fitness-1.csv" dataset. The dataset is stored within a PostgreSQL database as per the project requiremnts, and our NodeJS api provided in the backend folder allows for 3 GET requests that implement SQL to return data from the database.

  1. Ensure the backend api is started by running "node express.js" in the backend folder. This should connect to the database URL automatically assuming it is activiated.
  2. Select a certificate from the control below the map.
  3. Select the name of a person that holds that certificate or click on a marker on the map to view information about them.
  4. Select a new person or certificate.
